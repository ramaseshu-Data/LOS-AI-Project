# LOS-AI-Project
AI Powered Data Analytics for Predicting Hospital Length of Stay to Enhance Bed Capacity and Staffing Decisions

Project Overview

This project focuses on using Artificial Intelligence (AI) and data analytics to predict how long a patient is likely to stay in a hospital. The goal is not just prediction accuracy, but to show how these insights can help hospitals improve bed management, patient flow, and staffing decisions.

Hospitals often rely on past averages, which are not always reliable. This project demonstrates how data-driven models can provide more accurate and practical support for operational planning.

Objectives
Understand the concept of hospital length of stay (LOS)
Build predictive models using patient and clinical data
Analyse how different factors influence LOS
Evaluate model performance using standard metrics
Connect predictions to real-world hospital decisions like bed allocation and staff planning
Dataset

This project uses a publicly available dataset from Kaggle:

~100,000 patient records
Includes:
Demographic details (age, gender)
Clinical conditions
Lab results and vitals
Admission details
Length of stay (target variable)
Technologies Used
Python
Pandas, NumPy
Scikit-learn
Jupyter Notebook / VS Code
Matplotlib, Seaborn
Methodology

The project follows a structured machine learning workflow:

Data cleaning and preprocessing
Feature engineering and encoding
Train-test split for evaluation
Model building using regression techniques
Performance evaluation using:
Mean Squared Error (MSE)
Root Mean Squared Error (RMSE)
R² Score

Both simple baseline models and more advanced models are used to compare performance.

Key Insights
Patient condition, age, and admission type strongly influence LOS
Machine learning models can capture complex relationships better than traditional methods
Predictions can support:
Better bed capacity planning
Improved patient flow management
Smarter staff scheduling
Challenges
Data quality and missing values
Model interpretability (black-box problem)
Ethical concerns like bias and patient privacy
Integration into real hospital workflows
Future Scope
Use real-time hospital data
Integrate models into hospital decision support systems
Apply explainable AI for better transparency
Expand to multi-hospital datasets
Conclusion

This project shows that AI is not just about prediction, but about better decision-making. By combining analytics with hospital operations, it is possible to improve efficiency while maintaining quality of care.
