# LOS-AI-Project
  AI Powered Data Analytics for Predicting Hospital Length of Stay to Enhance Bed Capacity and Staffing Decisions
📌 Overview

This project uses AI and Machine Learning to predict hospital Length of Stay (LOS) based on patient, clinical, and admission data.

Instead of relying on rough historical averages, this approach demonstrates how predictive analytics can support:

🛏️ Bed capacity planning
👨‍⚕️ Staffing decisions
🔄 Patient flow optimization

The focus is not just on model accuracy, but on real-world hospital decision-making.

🎯 Key Objectives
Build predictive models for LOS using healthcare data
Identify key factors influencing patient stay duration
Evaluate model performance using industry metrics
Translate predictions into operational insights
📊 Dataset
Source: Kaggle (Microsoft LOS Dataset)
~100,000 patient records
Features include:
Age, Gender
Diagnoses & comorbidities
Lab values & vitals
Admission type
Length of Stay (target)
⚙️ Tech Stack
Category	Tools
Language	Python
Data Processing	Pandas, NumPy

Visualization	Matplotlib, Seaborn

🔍 Methodology
Data Collection → Data Cleaning → Feature Engineering → Model Training → Evaluation → Insights
Steps:
Data preprocessing (missing values, encoding, scaling)
Train-test split
Model building (Regression + Ensemble models)
Hyperparameter tuning (Grid Search + Cross Validation)
Performance evaluation
Metrics Used:
RMSE (Root Mean Squared Error)
MSE (Mean Squared Error)
R² Score
📈 Results & Insights
Clinical severity and admission type are strong predictors of LOS
ML models outperform simple statistical approaches
Predictions enable:
Better bed utilization
Reduced overcrowding
Efficient staff allocation
